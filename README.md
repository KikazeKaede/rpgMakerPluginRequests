# rpgMakerPluginRequests
RPGツクールを想定したプラグインの要望を投げると、時間があるときに作ってくれるようです。

# 依頼の仕方
Issues に 「プラグインの依頼はここでお願いします。」がありますので
そこにコメントをお願いします。
ショートカットはこちら：https://github.com/KikazeKaede/rpgMakerPluginRequests/issues/2

# 依頼したプラグインはいつできますか？
お約束するものではないので、いつ、できるかはわかりません。
もしできた場合は、本GitHubのリポジトリ上で公開します。
また、Xでご連絡いたします。

# 不具合があった場合は対応可能ですか？
お約束するものではないので、いつ、できるかはわかりません。
ご利用は計画的にお願いします。
但し、最低限のテストはしているつもりです。

# 有償で依頼を行いたいのですが？
現在は受け付けておりません。

# どうしてこんなことやっているんですか？
無理やりコーディングする時間を作るためです。
